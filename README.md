# SQL Exploratory Data Analysis

## Project Overview
This project explores the layoffs dataset using MySQL to identify trends, rankings, and business insights.

## Dataset
Layoffs dataset used for SQL exploratory data analysis.

## Analysis Performed
- Maximum layoffs and layoff percentages
- Companies with the highest layoffs
- Average layoffs by company
- Monthly layoff trends
- Rolling total of layoffs
- Year-wise layoffs
- Top 5 companies with the highest layoffs each year

## SQL Concepts Used
- Aggregate Functions (SUM, AVG, MAX)
- GROUP BY
- ORDER BY
- Common Table Expressions (CTEs)
- Window Functions
- DENSE_RANK()
- Date Functions

## Tools
- MySQL Workbench
